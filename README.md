## 🚀 Overview

**AuthAgent** is the first AI agent framework with native **Decentralized Identity (DID)** support for autonomous transaction signing. Built upon the Microsoft AutoGen framework, AuthAgent enables agents to possess their own cryptographic identity on-chain. Unlike traditional agents that rely on centralized API keys or human-signed transactions, AuthAgent agents generate a unique wallet address and register it on a smart contract registry.

This system implements a **'Capability Binding'** primitive where specific smart contract functions are bound to the agent's DID, allowing the agent to execute transactions autonomously while maintaining verifiable ownership. This solves the critical "who did it" problem in AI-driven DeFi.
